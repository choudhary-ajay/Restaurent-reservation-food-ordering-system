# 🍽️ Restaurant Reservation & Food Ordering System

A full-stack MERN application that allows customers to reserve restaurant tables, browse the menu, place food orders, and make secure online payments. The system also provides an admin dashboard to manage reservations, menu items, and customer orders efficiently.

## 🚀 Features

### Customer
- User Authentication (JWT)
- Browse Restaurant Menu
- Filter Food Items
- Add Items to Cart
- Place Food Orders
- Reserve Tables
- Select Date & Time for Reservation
- Online Payments (Stripe & Razorpay)
- Cash on Delivery (COD)
- View Order History
- Responsive Design

### Admin
- Secure Admin Login
- Add, Edit & Delete Food Items
- Manage Categories
- View & Manage Orders
- Update Order Status
- Manage Table Reservations

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

Admin

```bash
cd admin
npm install
```

---

## ▶️ Run the Project

### Backend

```bash
npm run server
```

### Frontend

```bash
npm run dev
```

### Admin

```bash
npm run dev
```
